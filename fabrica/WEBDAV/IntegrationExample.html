OfficeOpen can be easily integrated into your existing application to extend functionality with document editing.

You can use build-in or external web dav server.

Here are steps to integrate OfficeOpen into any html page:
1. Include config.js and OfficeOpen.js script files into html page.
<!--JavaScript configuration code-->
<script type="text/javascript" src="js/config.js"></script>
<!--main JavaScript code-->
<script type="text/javascript" src="js/OfficeOpen.js"></script>

2. Use openFile(fileName) method too pen file for view/edit.
fileName here is relative path to file on web dav server.
Note: Url to web dav server configured in config.js file with webdavUrl and webdavUrlPrefix properties.
Full url to file will be combined webdavUrl+webdavUrlPrefix+fileName.
