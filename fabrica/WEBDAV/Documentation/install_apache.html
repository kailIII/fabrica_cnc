OfficeOpen application install at Apache web server

Apache and PHP Installation

1. Apache Installation

  • Please refer to the official Apache documentation for detailed instruction on how to install Apache web server for your OS: http://httpd.apache.org/docs/current/en/install.html .

2. PHP Installation

  • Please refer to the official PHP documentation for detailed instruction on how to install PHP for APACHE on your OS: http://php.net/manual/en/install.php .

2. Enable mod_rewrite module

Apache by default comes with the mod_rewrite module installed but not enabled. If you have Apache installed on your own server, you should enable it .
  • Find the httpd.conf file (usually you will find it in a folder called conf)
  • Inside the httpd.conf file uncomment the line LoadModule rewrite_module modules/mod_rewrite.so (remove '#' sign in beginning of the line)

OfficeOpen Installation

3. OfficeOpen Installation...

  • Place OfficeOpen code on web server:
    • Open Apache server's root folder (various for different OS and apache installations /usr/local/apache/htdocs.)
    • Extract the OfficeOpen zip archive to a new WEBDAV folder inside the root folder. You can use any name for the folder. In this manual we use these diarectory and name as default.

 

Now you can test the OfficeOpen. Open the application URL in your browser, e.g. "http://localhost/WEBDAV/index.html". You should see a list of test files from "testfiles" subdirectory. Now you should configure OfficeOpen to get files from a necessary location.
