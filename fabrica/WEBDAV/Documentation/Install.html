OfficeOpen installation 

You can use build-in or external webdav server. If you decide to use external webdav server (already have configured one), then you should specify absolute url to the initial folder in config.js file with webdavUrl property and read integration guide.

If you decide to use webdav server build into OfficeOpen , please follow apropriate installation guide.

OfficeOpen installation is easy. In most cases you should simply extract distribution package to a folder on your web server.
