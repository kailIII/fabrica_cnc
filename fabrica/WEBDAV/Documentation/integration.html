OfficeOpen integration

OfficeOpen can be easily integrated into your existing application to extend functionality with document editing.

You can use build-in or external webdav server. If you decide to use webdav server build into OfficeOpen , please follow apropriate installation guide:
at install section. Perform configuration of control. If you decide to use external webdav server, then you should specify absolute url to the initial folder in config.js file with webdavUrl property.

Here are steps to integrate OfficeOpen into any html page:
1. Include config.js and OfficeOpen.js script files into html page.
<!--JavaScript configuration code-->
<script type="text/javascript" src="js/config.js"></script>
<!--main JavaScript code-->
<script type="text/javascript" src="js/OfficeOpen.js"></script>

2. Use OfficeOpen.OpenFileWith(fileName, mode) method to open file for view/edit.
fileName here is relative path to file on web dav server.
Note: Url to web dav server configured in config.js file with webdavUrl and webdavUrlPrefix properties.
Full url to file will be combined webdavUrl+webdavUrlPrefix+fileName.

<a href="#" onClick="javascript: OfficeOpen.OpenFileWith('Testdoc.docx', 0); "> Open file Testdoc.docx</a>

Java script methods

Parameter name Desciption Parameters
OfficeOpen.OpenFileWith(filePath, mode) Open specified file with associated application. Application priority determined by mode parameter.

filePath - path to file relatiove to webdav initial folder. Full url to file will be combined from following variables webdavUrl+webdavUrlPrefix+fileName.

mode - determine application priority.
Possible values:
0 - Use Microsoft Office firstly, if it not installed use OpenOffice.
1 - Use OpenOffice (or LibreOffice), if it not installed , use MS office.
2 - Use Microsoft Office. If not installed do nothing
3 - Use OpenOffice (or LibreOffice). If not installed do nothing.
